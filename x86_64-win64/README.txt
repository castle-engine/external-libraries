Here you will find DLLs for Windows 64-bit useful for programs
developed using the Castle Game Engine, [http://castle-engine.sourceforge.net/].

Origins:
- libpng and zlib: http://www.gtk.org/download/win64.php
  ("Third Party Dependencies", links to
  http://ftp.gnome.org/pub/GNOME/binaries/win64/dependencies/ )
